SpinWheel

A common problem in teaching is that if you pose a question to a class, the same few students, usually in the first row, answer them. These students are typically not representative of the class, being more advanced and also paying attention. To get around this problem, SpinWheel uses a spinning "prize wheel" image (+sound) to randomly select a name from a list. Once a name is selected, it is removed from the list (although this behavior can be suppressed).

Setup: Create a list of names from the class roster (usually a spreadsheet). You can use full names, or an abbreviated form, such as first name plus the inital of last name. Join the names together, with a comma or a semicolon as the separator, replacing any spaces in the names with plus (+) signs. Then generate a URL string that looks like this:
http://pyintro.org/wheel/?session=math101&names=John+Smith;Jane+Doe;Gilgamesh

The above URL can be stored in a file and reused to restart a session, as needed. Note that the query symbol (?) denotes the start of the parameter list and the ampersand (&) separates different parameters. The optional session parameter enables multiple parallel sessions. Another optional parameter max may be used to specify the maximum number of names displayed (default: 30). If there are more names than this maximum, a random subset of names is created and displayed. If the names parameter is omitted, the list of comma- or semicolon-separated names should be entered manually via a dialog box.

Usage: Click on SPIN to rotate the wheel. When a name is selected, a dialog box will appear. Clicking OK will remove the selected name from the list, clicking Cancel will retain the name.

Sessions: A sequence of spins starting with a new list of names is referred to as a session. The default session has a blank name. Alternate session names are specified using the optional session parameter in the URL and persist indefinitely in the browser (using information stored in browser local storage). Restarting a session will restore the original list of names. Clearing a session will open a dialog box to enter a new list of names.

Sample URLs:
http://pyintro.org/wheel/ Enter names via dialog box for default session
http://pyintro.org/wheel/?names=John+Smith,Jane+Doe Default session, comma-separated names
http://pyintro.org/wheel/?session=calc101&names=John+Smith,jr.;Jane+Doe Named session, semicolon-separated names, max. display of 15 names

Code: The source code is available on Github. To customize it, modify the www/index.html file and install in your own web site. (An undocumented option stats=1 can be specified to verify the randomness of the selection algorithm.) This implementation is based mostly on the jQuery rouletteWheel code.
