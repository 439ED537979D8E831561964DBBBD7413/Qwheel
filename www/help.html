Wheel Of Names

The Wheel Of Names uses a spinning "prize wheel" image (+sound) to select a name from a list. This can be used to randomly select students in a classroom to answer questions (with or without replacement of names).

Usage: Click on SPIN to rotate the wheel. When a name is selected, a dialog box will appear. Clicking OK will remove the selected name from the list, clicking Cancel will retain the name.

Setup: To populate the list, simply link to this site using an URL of the form:
http://pyintro.org/wheel/?names=John+Smith;Jane+Doe;Gilgamesh
with names separated by semicolons, with any spaces in the names replaced with plus (+) signs. The URL may be stored in a file and reused to restart a session. Alternatively, the list of semicolon-separated names may be entered manually via a dialog box. The optional session parameter enables multiple parallel sessions. Another optional parameter max may be used to specify the maximum number of names displayed for the final cut (default: 30).

Sessions: A sequence of spins starting with a new list of names is referred to as a session. Sessions persist indefinitely in the browser (using information stored in browser local storage). Restarting a session will restore the original list. Clearing a session will open a dialog box to enter a new list of names.

Code: This implementation is based mostly on the jQuery rouletteWheel code.
